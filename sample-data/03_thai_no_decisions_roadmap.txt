โน้ตประชุม — วางแผน Roadmap ไตรมาส 3 (โน้ตคร่าวๆ)
30/3/69

คนเข้าร่วม: ทีมโปรดักต์ทั้งทีม + ดีไซน์ 2 คน

- เปิดประเด็นว่าจะโฟกัสมือถือก่อน หรือทำ API redesign ก่อนดี
- พี่แซมว่าควรเป็นมือถือ บอกว่าการเติบโตอยู่ตรงนั้น
- พี่เลน่าแย้ง ลูกค้า enterprise ถามหาเรื่อง API มาตลอด
- เถียงกันไปมาเรื่องนี้อยู่นาน
- มีคน (ราช?) พูดขึ้นมาว่าเรายังไม่มีข้อมูล churn ของมือถือเลย ควรดึงมาดูก่อน
- ตกลงกันว่าต้องมีตัวเลข แต่ไม่มีใครรับว่าจะไปหามา??
- คุยเรื่องจ้าง dev มือถือเพิ่มอีกคน ไม่ได้ข้อสรุป ขึ้นอยู่กับงบที่การเงินยังไม่คอนเฟิร์ม
- ดีไซน์เอา mockup มาโชว์ ดูดี แต่ไม่ได้จดฟีดแบ็กไว้
- หมดเวลาพอดี
- "ไว้คุยต่อสัปดาห์หน้า" — แต่ไม่ได้นัดวันอะไรเลย

สรุปแล้วไม่แน่ใจว่าตกลงอะไรกันบ้าง รู้สึกว่าคุยเยอะแต่ไม่ได้ข้อสรุป
